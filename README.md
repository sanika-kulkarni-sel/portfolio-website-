# Personal Portfolio Website

A modern and responsive personal portfolio website built using HTML, CSS, and JavaScript. The portfolio showcases projects, technical skills, education, and contact information with a futuristic dark-themed user interface and interactive design elements.

---

## Features

- Responsive modern UI design
- Dark futuristic theme
- Interactive animations and hover effects
- Projects showcase section
- Skills section
- About Me section
- Contact form integration using EmailJS
- Downloadable resume section
- Smooth scrolling navigation

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
- EmailJS
- Font Awesome

---

## Projects Included

### SecurePass Tool
A cybersecurity-focused CLI utility that generates cryptographically secure passwords and evaluates password strength using custom security scoring logic.

### Glow Beauty Website
A modern responsive beauty product landing page with aesthetic UI, smooth layouts, and interactive frontend design.
